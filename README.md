# CodeSage AI 🚀

CodeSage AI is an AI-powered developer assistant that helps programmers understand, optimize, and improve their code. The application leverages Large Language Models to provide code explanations, bug analysis, and optimization suggestions through a simple and responsive user interface.

## 🌐 Live Demo

https://codesageai.vercel.app

## 📌 Features

* 🔍 Explain code in simple language
* ⚡ Suggest optimized versions of code
* 🐛 Identify potential bugs and issues
* 📖 Improve code readability and maintainability
* 🎨 Clean and responsive user interface
* 🤖 AI-powered responses using Groq API
* 💻 Supports multiple programming languages

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/NavaneethaSB/CodeSage-AI.git
```

### Navigate to Project Directory

```bash
cd CodeSage-AI
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
